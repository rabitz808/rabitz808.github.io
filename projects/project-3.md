This program will was for my ICS 212 class. I was tasked with creating a simple command line based program that would use math functions in C. The first thing I thought of creating was a program that would solve for the hypotenuse of a triangle given sides 1 and 2. Afterwards I decided I should add more to it by generating random side lengths for the triangle before calculating the hypotenuse. 

Upon opening the program, it will first calculate the hypothenuse of a right angle triangle with randomly generated lengths from 1-255. You can then type in your own values for each side to calculate the hypotenuse or find the sin, cos, and tan of radians. The functions used in this program are sqrt(), srand(), pow(), sin(), cos(), tan(). 

Learning to code in C was a lot harder than learning to code in Java. C is the second language I've learned a little bit about in ICS 212. This program was also my first attempt at using the srand() function to simulate randomness.
