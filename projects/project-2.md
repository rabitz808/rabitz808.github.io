The Chase is a simple game developed in Java in which the player has to collect 3 gold coins and make it to the other end of the map before the timer runs out. The challenge is that there are animals and cars that may hit you while you traverse the map. You start with 3 lives and if you get hit, you lose one life and get sent back to the start.

In this project I gained experience with using classes in Java and collaborating with other team members. Communication is key if you're working with other members, so we setup a discord server to share information and ideas we had when coming up with the game. Once we established our ideas we worked to implement code by distributing work based on the classes we would need for the game. I was in charge of setting up classes for spawning in the cars and animals as well as the player lives.

Aside from learning about how to use classes in Java, I also learned a lot about animations and sprite sheets. Making the character look like it was moving was just storing pictures in an array and displaying different images based on what key the player pressed. Overall, this was a fun project and I hope to make more in the future.
