## First Dive
When learning a programming language, everyone has to start somewhere. In my case, the first time I was introduced to a programming language was back in 2014. I was introduced to Lua through a game called Garry's Mod (Gmod) and although I never fully understood most of the code that I would see, I taught myself how to code just enough through trial and error. I would change a few lines at a time and run the game to see if there were errors. If there were any errors, I'd go back into the code and troubleshoot till I got it working. The feeling of sucessfully accomplishing my desired goal through the use of Lua was overwhelming. In my case, it was to convert models into playable characters and add them to a cosmetic shop for my Gmod server. 

## Success
The server I ran for Gmod was a hit. It supported a maximum of 27 players and was full everyday of the week. I started a steam group which is similar to a club for all of the players on my server. At one point, it had over 800 members. Learning to configure my server required me to also learn a little bit of MySql. It was linked to my server to store points for each player so they would retain their points everytime they logged back on.

## JavaScript
After learning a little bit of Lua, I was introduced to Java when I took my first ICS 111 course. The class was taught by Jason Leigh at UH Manoa. He is definetly one of my favorite professors with his funny humor and easy to follow along lectures. It gave me the foundation and basic knowledge I needed for programming in multiple languages. I'm really excited to learn JavaScript in ICS 314 this semester. I believe its a really useful language for Software Engineering and it seems to be slightly easier than Java. You don't need to declare types and the syntax is pretty similar to other languages. 

The freecodecamp website covered the basics of learning JavaScript. Although I said it seems easier than Java, I did struggle a little with the later topics. Most of my issues stemmed from the fact that its been a while since I've coded anything in general. I belive coding is a skill you can learn to be proficient at through lots of time and practice. Luckily we have workouts of the day (Wods) to supplement our learning in this class. I also love how its more focused towards web development as that's something I also really want to learn.
