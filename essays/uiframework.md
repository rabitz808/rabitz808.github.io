## Will This Really Benefit Me?
How much trouble have you had trying to set up a webpage and trying to format it exactly how you wanted it to look like? My guess is you've probably spent countless hours searching the web for specific classes you can edit for both html and css to make your webpage looking good. Now what if I told you that theres a much easier approach to get your desired outcome in a much shorter amount of time? Well, I'm here to tell you there is a way! The way I'm reffering to is to use whats called a ui framework. There are multiple different frameworks you can use out there that all come packed with documentation and examples to make your life easier. Think of it as using an api and trust me when I say its a life saver.

## Semantic UI is A Life Saver
The UI framework I've recently gotten familiar with is called Semantic UI. Its completley free to use and even has a course on pluralsight that you can take to learn all about the features it offers. I personally love how the framwork is similar to english syntax in the way that makes it easy to read and tweak. You can create a button class and specify the size and color all by using syntax similar to "ui tiny red button". They even provide a ton of icons you can use such as social media or generic user icons. The best part is it's all customizable and easy to do. You can even combine multiple icons to make a new one, how awesome is that?

## Why Learn a Framework When I Can Just Use Basic HTML and CSS??
By using a UI framework, you don't need to start from scratch! You can simply look up documentation on something you need like a dropdown menu. Then you can use the example code shown for the dropdown menu that matches your goals and tweak it to your liking. Doing it from scratch would just be a headache, especially if you're short on time. It can also help make projects easier if you need to collaborate with others. Working with a clean framework can increase production quality and everyone would be familiar with how to use it. Sure you need to spend time learning the framework, but once you've done some research you'll be an expert in designing what you want in a reasonable amount of time.
